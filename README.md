# Xtra Computing Group Blog

The official blog of [Xtra Computing Group](https://github.com/Xtra-Computing) at NUS, powered by [Docusaurus](https://docusaurus.io/).

**Live site:** https://Xtra-Computing.github.io/blog

## Contributing

Everyone is welcome to contribute! See [CONTRIBUTING.md](./CONTRIBUTING.md) for how to write and submit a blog post.

## Local Development

```bash
npm install
npm start
```

The site will be available at `http://localhost:3000/blog/`.

## Deployment

Merging a PR into `main` automatically triggers GitHub Actions to build and deploy to GitHub Pages.
